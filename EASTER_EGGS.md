## Tips

- Word triggers work **anywhere** as long as you're not focused on an input field
- The Konami code uses arrow keys then lowercase `b` then `a`
- Your found eggs are tracked in `localStorage` under key `ee_found`
- The idle screensaver resets when you move the mouse or press any key
- Mouse shake requires ~9 quick direction changes within 1 second
- Corner clicks must be in order: top-left → top-right → bottom-right → bottom-left
- Check the **browser console** on load for a hidden hint 😉
